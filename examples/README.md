# Examples

The committed example artifacts are generated from the deterministic `mock:perfect`
baseline against benchmark set `v1`:

- `sample_run.json`: machine-readable run trace
- `reports/mock-perfect-report.md`: readable case-by-case report
- `reports/mock-perfect-report.html`: standalone visual report
- `screenshots/`: dashboard captures

Regenerate them after report or benchmark changes by running a perfect mock benchmark and
rendering its latest run through the report writers.
